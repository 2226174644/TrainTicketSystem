# TrainTicketSystem
火车票售票系统-数据库课设
系统整体采用QtCreator + PostgreSQL编写。利用Qt来快速方便的绘制UI界面，做出较好的交互界面。用ODBS与PostgreSQL数据库进行连接，使用Servlet技术，在表单提交的基础上与使用者进行信息交互，对数据库中已有数据进行查询、增添、删除或修改等操作，完成项目的主要功能。
